## How To Install

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
  
## Technologies

Project was created with:

- next.js
- swiper
- photo album
- lightbox

## Functionalities

- gallery with swiper, photo album and lightbox combined.
